# 02 修炼境界体系 / Cultivation Realms

> 经典五境界：**炼气 → 筑基 → 金丹 → 元婴 → 化神**。
> 设计原则（见 [00_overview](00_overview.md) 支柱 3）：每境解锁**新机制**，不只是数值堆叠。

## 境界总表 / Realm Overview

| # | 境界 | en_us | 一句话 | 核心解锁机制 |
|---|------|-------|--------|--------------|
| 1 | 炼气 | Qi Refining | 引气入体，洗练凡躯 | 感应/吸纳灵气、辟谷雏形、轻微强身 |
| 2 | 筑基 | Foundation Establishment | 筑就道基，蜕凡为修 | 完全辟谷、御物雏形、第一法术、抗污染 |
| 3 | 金丹 | Golden Core | 凝气成丹，法力外放 | 御剑/御物远程、稳定法术、丹田储法力 |
| 4 | 元婴 | Nascent Soul | 元婴出窍，神识大成 | 神识扫描、出窍探索、重伤不死（元婴遁逃） |
| 5 | 化神 | Spirit Severing | 神游太虚，半步超脱 | 域内法则微操、强力法术、近乎辟易凡敌 |

> 每个大境界内部细分**小层**（如炼气一层~九层、筑基初/中/后期等），小层走数值与进度积累，大境界突破才解锁机制。
> **进度载体（定）**：每个大境界 = 一个 spell，spell 的 **level** 即小层进度（见 [03_qi_system](03_qi_system.md) 第三节）。具体分层见各境小节。

## 境界一：炼气 / Qi Refining

**主题**：从凡人到「能感应并容纳灵气」的入门者。这是整个 M1 里程碑的核心。

- **入门条件**：灵根相对清晰（幸存即有灵根，见 lore）+ 一次「感应灵气」的契机（残卷/灵材/顿悟）。
- **分层**：炼气一层 ~ 九层（「炼气」spell 的 level，线性积累灵气容量与吐纳效率）。
- **修炼方式**：打坐吐纳吸纳环境灵气，或炼化灵材/妖丹补充。
- **解锁机制**：
  - 感应灵气（能看到环境灵气浓度概念）。
  - 缓慢自动恢复少量灵气。
  - 轻微强身（小幅属性/抗性，体现「洗练凡躯」）。
  - **辟谷雏形**：饥饿/口渴速度略降〔待平衡〕。
- **风险**：强行冲层/突破时有反噬风险（事件性，见下文「跨境界通用设定」）。

## 境界二：筑基 / Foundation Establishment

**主题**：道基既成，脱离凡俗体质。M2 里程碑核心。

- **突破条件**：炼气圆满 + 足量灵气积蓄 + 突破时抵御反噬（一次有风险的「突破」事件）。
- **分层**：初期 / 中期 / 后期。
- **解锁机制**：
  - **完全辟谷**：不再需要进食（或极大降低），重塑生存玩法。
  - **御物雏形**：可短程隔空取物/操控轻物。
  - **第一个主动法术**（接驳 dda SPELL，详见 04）。
  - **强健体魄**：对辐射/疾病/环境危害的抗性显著提升（道基既成，百毒难侵）。
- **风险**：筑基失败可能重创甚至废修。

## 境界三：金丹 / Golden Core

**主题**：灵气凝结成丹，法力可外放成术。

- **突破条件**：筑基圆满 + 凝丹机缘（高品质灵材/丹药）。
- **解锁机制**：
  - **金丹储灵气**：灵气上限与稳定性大增，丹田成为「灵气池」。
  - **御剑/御物远程**：远程攻击/移动手段。
  - 法术体系成型（多个可选流派的雏形，见 04）。
  - 寿元延长（叙事层）。

## 境界四：元婴 / Nascent Soul

**主题**：凝婴成功，神识大成，半步脱离肉身束缚。

- **解锁机制**：
  - **神识扫描**：大范围探测（敌人/资源/地形），接驳地图/侦测机制。
  - **元婴出窍**：可短暂脱离肉身探索（高风险高回报）。
  - **元婴遁逃**：肉身被毁时元婴可逃，避免直接死亡（一种「续命」机制）〔待平衡，需防滥用〕。

## 境界五：化神 / Spirit Severing

**主题**：神识与天地法则初步交感，半步超脱，本 mod 当前的终局境界。

- **解锁机制**：
  - 局部法则微操（强力 AOE/控制类法术）。
  - 对凡俗敌人近乎碾压（但高阶妖兽/异界存在仍具威胁，维持挑战）。
  - 通往「飞升」的叙事钩子（远期，未必实装为可达终点）。

## 跨境界的通用设定 / Cross-Realm Notes

- **瓶颈与突破**：每个大境界圆满后需要「突破」，是一次有失败风险的事件，可能消耗稀有资源、触发反噬。突破是里程碑节点，要有仪式感与紧张感。
- **机制优先于数值**：属性加成保持克制；让玩家因为「能辟谷了/能御剑了/能神识扫描了」而兴奋，而不是因为 +5 力量。
- **与原版接驳（关键决策已定，详见 [03_qi_system](03_qi_system.md) 与 `09_integration` 待写）**：
  - 灵气 → 独立自定义 vitamin 能量池（不复用 int 绑定的 mana）。
  - 境界 → `mutation` 形式的「境界标记」突变，每境一个，互为前置。
  - 小层进度 → **每个大境界一个 spell，用其 level 表征**。
  - 解锁的能力 → 主动神通用 `SPELL`（能量源指向灵气 vitamin），被动用 mutation 的 enchantment。
- **数值全部待平衡**：本文出现的所有效果强度均为方向性占位，标 〔待平衡〕，留到 `08_progression_balance` 统一拉表。
